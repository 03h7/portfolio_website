Twitch chat bot

Un "bot" connecté au chat IRC de la chaîne Twitch.tv d'un utilisateur.
Le programme va pouvoir intercepté les messages envoyés dans le chat pour ensuite exécuter certaines actions en fonction du contenu du message.


Fonctionnement du bot :

  • Un utilisateur saisit une commande. (généralement un mot-clé précédé d'un " ! ")
  • Le bot vérifie que la commande n'est pas en délai.
  • Exécuter une action, comme par exemple donner la durée du stream.
    Dans mon cas, le bot répond à l'utilisateur ayant saisi la commande avec un message sélectionné selon un nombre aléatoire pondéré.
  • En fonction du message séléctionné, un certain nombre de points est attribué à l'utilisateur.
  • Les points de l'utilisateur sont mis à jour dans la base de données grâce à une requête SQL.

Ensuite, j'ai mis en place un site web lié à la même BDD pour afficher le classement des utilisateurs.
Le site est composé d'une page, qui affiche le classement triant les utilisateurs du plus haut score au plus bas.
La date et l'heure correspondant à la dernière utilisation de la commande sur le chat est aussi renseigné sur le site.

technologies utilisées
pour le bot : tmi.js / node.js / postgreSQL / heroku (hébergement du service)
pour le site : node.js / express / ejs / postgreSQL / heroku (hébergement du site)
